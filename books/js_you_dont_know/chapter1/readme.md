##    何为作用域链

    函数是一等对象
    每一个函数执行时，会在自己的创建的AO对象上找对应的属性值
    AO 函数的管理：若找不到则往父函数AO上找,一直到Windows(全局作用域)
    AO链就是JavaScript中的作用域链
    
    编译阶段 AO生成 

AO上如果有与函数同名的属性的，则会被此函数覆盖
